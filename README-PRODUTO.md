## 🚀 INSTALAÇÃO RÁPIDA

### Requisitos
- Node.js 18+
- PostgreSQL 12+ (local) OU
- Conta Supabase (nuvem)

### Windows
```bash
1. Execute install.bat
2. Escolha PostgreSQL ou Supabase
3. Siga as instruções
4. Acesse http://localhost:3000
```

### Linux/Mac
```bash
1. Execute chmod +x install.sh && ./install.sh
2. Escolha PostgreSQL ou Supabase
3. Siga as instruções
4. Acesse http://localhost:3000
```

---

## 🔧 CONFIGURAÇÃO

### PostgreSQL Local
1. Instale PostgreSQL
2. Execute o script de setup
3. Configure em `config.postgresql.json`

### Supabase Nuvem
1. Crie conta em supabase.com
2. Execute SQL em `database/supabase-setup.sql`
3. Configure em `config.supabase.json`
